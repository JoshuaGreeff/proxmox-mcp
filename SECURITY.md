# Security Policy

If you find a vulnerability in `proxmox-mcp`, do not open a public issue with exploit details or live credentials.

Report it privately through GitHub Security Advisories for this repository. If that path is unavailable, contact the maintainer directly and include:

- affected version or commit
- impact summary
- reproduction steps
- any required configuration details with secrets redacted

Please avoid testing against systems you do not own or operate. For disposable lab validation, use least privilege and clean up any temporary resources you create.
